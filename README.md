# JoinMessageReloaded
 Customize your Join/Quit Message
# Features:
Customize Join/Quit Message with Custom Text and Colours.
The Messages are shown, when starting the server.
You can choose if Players get a private Message when Joining.

# Installation:
Just drag the .jar in your Server directory and reload the server.

IMPORTANT:
If you found a bug or something is not workin dont use the rewiev function! Instead, please create a ticket here: https://github.com/LeNinjaHD/JoinMessageReloaded/issues
